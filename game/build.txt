инструкция по сборке проекта

===============================================================================
   Компиляция

необходимые инструменты
 1. Microsoft Visual Studio 2008 SP1
 2. Microsoft DirectX SDK
 3. Perl
 4. aoTuV encoder

Первым делом заходим в папку make и запускаем скрипт 'sound.pl'.
 (происходит конвертация всех wav в ogg)

===============================================================================
   Сборка и распространение релиза

Сначала нужно откомпилировать Release версию проекта (см выше).
После этого исполняемые файлы окажутся в папке 'out/Release'.
Остальные файлы, необходимые для работы игры, лежат в папке 'workdir'.

Заходим в папку make и запускаем скрипт 'package.pl'.
Создастся папка 'out/Tank' и в нее будут скопированы все необходимые файлы.
Если папка 'out/Tank' уже существует, то сначала ее нужно удалить.

Для упаковки игры рекомендуется пользоваться архиватором 7-zip, так как он
обеспечивает наилучшее сжатие.
